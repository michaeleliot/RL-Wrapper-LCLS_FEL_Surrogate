# LCLS_FEL_Surrogate_RL
RL Wrapper around the FEL Surrogate Model

## Install
Create Venv, Activate Venv, Install from requirements.txt
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Train the RL Model
```bash
python rl_train.py
```

## Evaluate the RL Model
```bash
python rl_eval.py
```
